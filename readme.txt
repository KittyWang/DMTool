多源海量动态数据挖掘工具
---------------------------------------------------
双击start.bat启动程序。

吴张算法运行：
1.点击“浏览”，选择dataset/wuzhang/下面的文件。点击“提交”。
2.重复步骤1，可以选择多个文件提交。
3.点击“确定”运行算法。

数据流分类算法
1.训练集处，点击“浏览”，选择dataset/driftDetect/下面的train文件。
2.数据流文件，点击“浏览”，选择dataset/driftDetect/下面的test文件。
3.参数可使用默认，点击“确定”运行算法。
4.通过“可视化”可查看决策树。“弹出”可弹出当前决策树，选择新的决策树展示并对比。

增量挖掘算法
1.点击“浏览”，选择dataset/下面的momenttest文件。
2.点击“确定”运行算法。
